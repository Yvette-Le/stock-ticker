# STOCK TICKER APP
Users will enter a valid stock symbol in the search form. The most recent data available will render in the display.
- Examples: IBM, BABA, AAPL

### API
All information regarding the stock data was gathered from https://www.alphavantage.co/documentation/

### Parcel Bundler
__.Babelrc__  
Babel is used to create a backwards compatible version of Javascript. In  this case, we are using it for the async function. The babelrc file can be found in the root file of this project.
